## Neon Recall - Gamified Dark Flashcards

Interactive flashcard app with:

- Dark neon UI with colorful gradients and animated ambiance
- Click-to-flip and keyboard-flip card interactions
- Gamified progress: XP, levels, streaks, best streak, mastery bar
- Review grading buttons: Again / Hard / Good / Easy
- Badge unlock system for milestones
- Deck controls: previous, next, shuffle, and full progress reset

## Run

Because this is a static app, you can run it in two easy ways:

1. Open `index.html` directly in a browser, or
2. Serve the folder locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Controls

- `Space`: flip card
- `Arrow Left / Arrow Right`: previous or next card
- `1`: Again
- `2`: Hard
- `3`: Good
- `4`: Easy
